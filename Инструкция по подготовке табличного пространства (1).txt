Выполнять задания можно на сайте https://livesql.oracle.com/ . 
Для получения доступа к функционалу требуется регистрация. После регистрации перейти на указанный сайт, откроется окно Learn and share SQL. Нажать на Start Coding Now. 
В открывшемся SQL Worksheet можно создать необходимые таблицы. Скрипт для создания нужных таблиц прикладываю - script.sql. Вставьте все конструкции из файла script.sql и запустите. Автоматически создадутся и заполнятся таблицы, нужные для выполнения лабораторных. Это необходимо выполнять при каждом новом входе в Live SQL, так как таблицы действительно только во время вашей текущей сессии. Если сессия закрывается по истечению времени, то таблицы автоматически удалятся.
Для написания запросов из лабораторной работы конструкции из script.sql можно удалить из SQL Worksheet и начать писать запросы.
